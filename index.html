Data Visualization techniques

This project includes the different types of techniques employed to visualize the data. Click on the each button to see the visualization.

  1. Developing visualizations in d3.js in three different ways: (1) “by hand”, (2) through Javascript without any libraries, and (3) using d3.js. Dataset: Scores [SATM, SATV, ACT, GPA]

    Visualization 1 - by hand

    Visualization 2 - through Javascript without any libraries

    Visualization 3 - using d3.js


  2. Create Brushable Scatterplot Matrix (also sometimes abbreviated as “SPLOMs”). Dataset: Iris [sepalLength, sepalWidth, petalLength, petalWidth, species]

  3. Creating interactive Parallel Coordinates visualization for multidimensional datasets. Dataset: Cars

  4. Visualization of hierarchical data using a technique known as Treemaps.

  5. Visualize three-dimensional scalar fields through the use of volume rendering. We will rely on a volume renderer from the VTK library.

  6. Basics of vector field visualization through encoding per-position properties with a variety of techniques. Specific objectives include:
    1. Using color and glyphs to encode both magnitude and orientation.
    2. Using streamlines to capture integrated properties of streamlines.
    3. Experimenting with both random and uniform seeding patterns to adjust the position of both glyphs and streamlines
